Enable theme switching for admin users.

== Installation ==

You can download and install Theme Switcher using the built in WordPress plugin installer. If you download Theme Switcher manually, make sure it is uploaded to "/wp-content/plugins/theme-switcher/".

Activate Theme Switcher in the "Plugins" admin panel using the "Activate" link. If you're using WordPress Multisite, you can choose to activate Theme Switcher network wide for full integration with all of your sites.
